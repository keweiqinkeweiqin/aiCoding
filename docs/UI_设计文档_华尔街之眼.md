# 华尔街之眼 — UI 设计文档

> 组41 | 版本：v1.0 | 日期：2026-03-26  
> 前端技术栈：Vue 3 + Element Plus + ECharts

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| 信息密度优先 | 投研场景用户需要快速扫描大量信息，布局紧凑但不拥挤 |
| 数据可视化驱动 | 关键指标图表化，减少阅读负担 |
| 信任感设计 | 通过颜色编码、置信度进度条、验证标记建立信息可信度感知 |
| 专业金融风格 | 深色主题为主（可选），参考 Bloomberg Terminal / TradingView 风格 |
| 响应式适配 | 桌面端优先，兼顾平板（路演可能用投屏） |

## 二、全局设计规范

### 2.1 色彩体系

```
主色调（品牌色）：
  - 主色：#1A73E8（科技蓝）
  - 辅色：#00C9A7（增长绿）

语义色：
  - 涨/利好：#E53935（红色，A股习惯）或 #00C853（绿色，美股习惯）→ 建议做配置项
  - 跌/利空：#00C853 或 #E53935
  - 权威 AUTHORITATIVE：#4CAF50（绿色）🟢
  - 一般 NORMAL：#FF9800（橙色）🟡
  - 存疑 DOUBTFUL：#F44336（红色）🔴

置信度色阶：
  - 0.0~0.3：#F44336（低置信）
  - 0.3~0.6：#FF9800（中置信）
  - 0.6~0.8：#2196F3（较高置信）
  - 0.8~1.0：#4CAF50（高置信）

情绪指数色阶：
  - 极度恐慌(0-25)：#B71C1C（深红）
  - 恐慌(25-45)：#E53935（红）
  - 中性(45-55)：#9E9E9E（灰）
  - 贪婪(55-75)：#43A047（绿）
  - 极度贪婪(75-100)：#1B5E20（深绿）
```

### 2.2 字体规范

```
标题字体：PingFang SC / -apple-system / Segoe UI, 16-20px, font-weight: 600
正文字体：PingFang SC / -apple-system / Segoe UI, 14px, font-weight: 400
数据字体：DIN Alternate / Roboto Mono（等宽，用于数字/股价）, 14-24px
辅助文字：12px, color: #999
```

## 四、关键交互流程

### 4.1 新用户首次使用流程

```
打开产品 → 欢迎弹窗（产品介绍 3 步引导）
    ↓
引导设置用户画像（关注领域 → 投资偏好 → 添加持仓）
    ↓
画像保存成功 → 跳转首页
    ↓
首页展示个性化排序的情报流（"与我相关"默认排序）
    ↓
点击某条情报 → 查看详情 + AI研判
```

### 4.2 AI 研判生成流程

```
用户点击"AI研判"按钮
    ↓
检查是否已设置画像 → 未设置则弹窗引导
    ↓
显示 loading 状态（骨架屏 + "AI分析师正在研判..."）
    ↓
后端调用 LLM 生成研判（SSE 流式返回）
    ↓
前端逐段渲染研判结果（打字机效果）
    ↓
生成完毕，显示"重新生成"和"切换视角"按钮
```

### 4.3 情报筛选与排序

```
默认展示：按时间倒序，优先展示与用户画像相关的情报
    ↓
用户可操作：
├─ 按来源类型筛选（新闻/社交/行情）
├─ 按领域标签筛选（AI芯片/云计算/...）
├─ 按可信度筛选（仅看权威/隐藏存疑）
├─ 按验证状态筛选（仅看已验证）
└─ 排序切换（最新/置信度最高/与我最相关）
```
